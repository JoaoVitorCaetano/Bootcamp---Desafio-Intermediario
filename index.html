Projetos Acadêmicos e Profissionais

Gerenciador de tarefas:

Um simples gerenciador de tarefas onde é possível adicionar, atualizar o status e excluir tarefas. Com funcionalidades de adicionar uma tarefa, criar novas tarefas informando qual a tarefa deve ser adicionada, excluir uma tarefa e atualizar status da tarefa: marque uma tarefa como concluída, adicione uma dificuldade e uma data de término, filtre as atividades por dificuldade e/ou data. Tecnologias Utilizadas JavaScript, HTML e CSS

link do repositório: https://github.com/JoaoVitorCaetano/Gerenciador-de-Tarefas.git

Gerador de Senhas:

um gerador de senhas que permite ao usuário escolher o tamanho da senha e copiá-la para a área de transferência. Funcionalidades: Escolha do tamanho da senha, geração de senhas seguras e aleatórias, com também a opção de copiar a senha gerada com um clique. Tecnologias Utilizadas: HTML, CSS e JavaScript. Como Usar: Defina o tamanho da senha desejada, clique no botão para gerar a senha e clique na senha para copiá-la para a área de transferência.

link do repositório: https://github.com/JoaoVitorCaetano/Gerador-De-Senhas.git
